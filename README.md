# Nguyễn Thành Trung - Business Analyst Portfolio

Đây là website portfolio cá nhân được xây dựng bằng HTML, CSS và JavaScript thuần, phục vụ định hướng nghề nghiệp Business Analyst.

## Nội dung website

Website gồm các phần chính:

- About Me
- Skills
- Certifications
- Projects
- Contact

Portfolio tập trung vào định hướng Business Analyst, Data Analytics và Cloud Data, đồng thời trình bày các chứng chỉ LinkedIn Learning và Google Cloud đã hoàn thành.

## Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Cách chỉnh sửa thông tin cá nhân

Mở file `index.html` và tìm các nội dung cần cập nhật:

- `your.email@example.com`: thay bằng email thật
- Link LinkedIn: thay bằng link LinkedIn cá nhân
- Link GitHub: có thể thêm vào phần Contact hoặc Project
- Các dự án: thay link repository GitHub của từng project
- Thông tin trường/khoa: chỉnh lại nếu muốn ghi đầy đủ hơn

## Cách deploy lên GitHub Pages

### Cách 1: Repository tên `username.github.io`

1. Tạo repository mới trên GitHub với tên:

```text
username.github.io
```

Trong đó `username` là username GitHub của bạn.

2. Upload file `index.html` và `README.md` lên repository.

3. Vào trình duyệt và mở:

```text
https://username.github.io
```

Website sẽ tự động hiển thị.

### Cách 2: Repository portfolio riêng

1. Tạo repository mới, ví dụ:

```text
ba-portfolio
```

2. Upload file `index.html` và `README.md`.

3. Vào `Settings` → `Pages`.

4. Ở mục `Build and deployment`, chọn:

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

5. Bấm `Save`.

6. GitHub sẽ tạo link dạng:

```text
https://username.github.io/ba-portfolio/
```

## Gợi ý ảnh minh chứng cho tiểu luận

Sau khi deploy, nên chụp các ảnh sau để đưa vào phụ lục tiểu luận:

- Trang chủ portfolio
- Mục Skills
- Mục Certifications
- Mục Projects
- Trang Settings → Pages chứng minh đã deploy thành công
